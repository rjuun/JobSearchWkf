✨ The one-line

Before, the coach sorted by category, so a fresh user saw a wall of near-identical "add ATS variants" cards. Now one value score ranks every prompt across engines: a live screening watch-out and a Core requirement on a role you're chasing rise to the top, and the six ATS cards collapse into one tidy multi-skill pass.

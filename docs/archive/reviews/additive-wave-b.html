Additive Plan · Wave B — additive migrations

Two small schema adds, no existing behaviour changed · ✓ Applied ⚑ flag-gated

✨ The one-line

Four new lenses now bolt onto the shipped app — a Statement of what you did, a Returns panel that tracks applications and their outcomes, a Coverage Matrix of your targets' must-haves, and a Sourcing Compass that ranks your capture channels by the fit they produce. Nothing was removed. Two columns were added (job_leads.source, an activity_events table); is_target and the applications table already existed and were finally put to work.

✅ What shipped (Wave B)

B1
Events log → the Statement. New activity_events table; existing server actions append one line each (evidence kept, target flagged, role screened, CV made, applied, outcome). A read-only /statement page re-projects the last 30 days into a re-entry ritual — "here's what you did." Monthly email deferred — in-app for now.
B2
Application outcomes → Returns. Downloading a tailored CV now opens an applications row (idempotent, best-effort); the dashboard Returns panel logs response / interview / offer / screened-out and nudges stale ones. Tracking first — no "lessons" until n is honest.
B3
Coverage Matrix tab. A new lens beside the strength meter (/profile/coverage): each target's Core/Important requirements as covered/gap cells. The meter stays; a ⚑ gap deep-links to the coach prompts that close it.
B4
Source → Sourcing Compass. A free-text source on capture (paste form + bookmarklet both pass it through); a panel on the board ranks channels by average fit produced, so a noisy alert becomes visible and retireable.

🔭 How to see it yourself

  1. Statement: keep evidence / flag a target / screen a role, then open Statement in the nav.
  2. Returns: download a tailored CV → it appears on the Dashboard → Returns panel; log an outcome.
  3. Coverage: open Career Graph → Coverage matrix (3 targets seeded) → click a ⚑ gap to jump to the coach.
  4. Compass: capture a lead with a Source (e.g. "LinkedIn alert") → the compass appears under the board table.

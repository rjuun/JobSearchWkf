# docs/archive — frozen build-log

Historical review artifacts from the prototype's construction. **These are a record of how the
system was built, not the current spec.** For the authoritative, up-to-date documentation, go one
level up to [`docs/`](../).

Nothing here is maintained — pages reflect the state at the time each wave shipped. Screenshots and
HTML review pages are kept for provenance and the demo narrative.

## What's here

| Folder / file | What it is |
| --- | --- |
| `phases/` | Per-phase write-ups (P1–P5), onboarding O2–O4, design-system + user-journey reviews, and the review-hub `index.html`. |
| `milestones/` | Milestone specs M0–M7 (the journey-first redesign build). |
| `reviews/` | Per-milestone + per-wave code-review reports (M0–M7, Additive waves A/B/C, phase A/B/C reports). |
| `rethink/` | Rethink-Completion wave reports (D/E/F) + the external Grok-review verification. |
| `reports/` | The 2026-06-30 "current prototype" roots report + screenshots. |
| `roots-review/` | Early roots-board review screenshots + index. |
| `roots-review-html-update-plan.md` | A one-off plan for updating the roots-review HTML. |
| `RoleProof_Rethink_Completion_Milestones.md` | The source plan for the Rethink-Completion waves. |

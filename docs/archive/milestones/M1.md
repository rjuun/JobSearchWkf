## Design decisions (reconciling the two source docs)

- **The "~60" target is really "mid-band with headroom".** This rich seed legitimately maxes Foundation
  and Freshness, so it lands at 72 (upper "Solid start"), not literally 60. The "Solid start" band was
  widened to 50–79 so the seed reads comfortably mid-band. The important properties — not saturated,
  large visible headroom (28), and a meter that *moves* — all hold.
- **Headroom, not loss (milestones doc) vs. "flagging lowers strength" (Phase 1 doc).** Resolved in favor
  of the milestones doc: relevancy earned depends only on *covered* requirements, so flagging a new target
  raises the **ceiling** (unlocks headroom) without ever dropping the number. Verified by a unit test.

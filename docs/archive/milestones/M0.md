## Remaining gaps / known interim state

- **Strength reads ~89/"Strong", not ~60/"Solid".** The current `strengthOf` has a hard floor of ~60
  (identity + positions + stories + edu/lang + bullets, all maxed by the rich seed). Believable gaps
  can't pull it into mid-band. **M1** delivers the component model whose Relevancy + Freshness
  dimensions sit empty for a fresh seed, creating the headroom that lands the same seed at ~60.
- Coach queue is still **tier-first** (ATS-type prompts group first) — cross-engine value ranking is **M2**.
- Orphan `.storage/cv-output/<old-lead-id>/` dirs from prior sessions remain on disk (harmless; not served — lead ids are regenerated each reseed). Not committed (`.storage` is gitignored).

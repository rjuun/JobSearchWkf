- UK

## Purpose

Perform a quick review of the Job Posting to identify any specific instructions or constraints regarding the application documents (CV and Motivational Letter). Special attention is given to the use of a Professional Headshot, considering both explicit instructions and implicit market or corporate expectations.

This step ensures that the application package complies with formal requirements and avoids common cultural or corporate missteps before tailoring begins.

---

## A. Execution Sequence

1. Open the Job Posting (LinkedIn or company careers page).
2. Scan the full text for any explicit instructions regarding:
    - CV format or structure
    - CV length (e.g. maximum number of pages)
    - File format (PDF, Word, etc.)
    - File naming convention
    - Language of the application
    - Requirement for a Motivational Letter / Cover Letter
3. Specifically analyse instructions and context around the use of a **Professional Headshot**.
4. Cross-reference with known market practices of the country where the Talent Acquisition team is based.
5. Record findings in the output table (see D.1).

---

## B Output

### Table – Application Format Check

Produce one table per Job Lead with the following structure:

| Check Item                          | Finding in Job Post                                                                | Recommended Action                                                     | Notes / Source                  |
| ----------------------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------- | ------------------------------- |
| CV Length                           | e.g. “max 2 pages” or not mentioned                                                | Follow instruction or use standard (2 pages)                           | Job Post                        |
| File Format                         | e.g. PDF only                                                                      | Use required format                                                    | Job Post                        |
| File Name                           | e.g. specific naming convention                                                    | Follow if specified                                                    | Job Post                        |
| Motivational Letter Required        | Yes / No / Not mentioned                                                           | Prepare or skip accordingly                                            | Job Post                        |
| Professional Headshot               | Explicitly requested / discouraged / not mentioned                                 | Include / Do not include / Optional                                    | Job Post + Market Practice      |
| Market Practice (Headshot)          | Country: [Country]                                                                 | Align with local custom                                                | External knowledge              |
| Corporate / DEI Considerations      | Any signals in Job Post or company                                                 | Adjust decision accordingly                                            | Job Post + Company research     |
| Check Item                          | Finding in Job Post                                                                | Recommended Action                                                     | Notes / Source                  |
| **HR / Talent Acquisition Contact** | Name and role if mentioned (e.g. “Saradha Vathana Selvendran – People Engagement”) | Note the name for potential personalization in the Motivational Letter | Job Post / Company careers page |


**Headshot Decision Guidance**

| Situation                                     | Recommendation                  | Reason                             |
| --------------------------------------------- | ------------------------------- | ---------------------------------- |
| Explicitly requested in Job Post              | Include                         | Follow instruction                 |
| Explicitly discouraged or forbidden           | Do not include                  | Follow instruction                 |
| Not mentioned + country is UK, IE, DK, NL, CA | Do not include                  | Strong DEI norms against headshots |
| Not mentioned + other European country        | Optional (lean towards exclude) | Increasing DEI sensitivity         |
| Not mentioned + company has strong DEI focus  | Do not include                  | Corporate alignment                |
| Strongly traditional / conservative industry  | Include (if appropriate)        | Cultural fit                       |


---

## Notes

- This is a **quick check step** and should not take more than 5–10 minutes per Job Lead.
- The main goal is to avoid easy disqualifications due to formal non-compliance.
- Findings from this step should be considered when preparing the final application package (CV + Motivational Letter).

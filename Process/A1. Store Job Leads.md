A Power Automate Flow ([Create Job Lead from LinkedIn URL](https://make.powerautomate.com/environments/Default-cfa77709-86a8-41a7-86b6-d487977d02b0/flows/29a8a29b-5d51-4de6-b95f-6ac33cb8ca8d/details?v3=false)) was created to perform the storing of Job Descriptions in Sharepoint.

The Flow is triggered when a HTTP Request is received.
HTTP Request is created by the Candidate when, being on the Site of the Job Description, he presses the a Edge Bookmark named "LinkedIn" which consists of the following JavaScript

This script captures the information about the Job Post and send the HTTP Request to the Power Automate Flow which will 
- Split the information in the proper fields
	- Title
	- URL
	- JD Text
- Create a new Job Lead in the [Job Leads](https://nunonine.sharepoint.com/sites/JobHunting/Lists/Job%20Leads/AllItems.aspx) list
- Take 2 chunks from the Bookmarklet and to bring them together in one
- Save the concatenated chunks into an Obsidian .md note
- Set the Process Status as: `Job Lead`
- Update the excel

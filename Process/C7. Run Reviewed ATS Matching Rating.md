Run Reviewed ATS Matching Rating for [Job Title] at [Company]  
  
  
1. Input Documents  
- Use the final tailored CV (the PDF/DOCX I just uploaded or the latest version).  
- Use the "JD Requirements Assessment" table we produced in Step 2 (the one with columns: Requirement_Group, Rank, Requirement, Requirement_Description, Skills, Connection to Areas of Expertise).  
  
  
Overall ATS Matching Rating  
Calculate a single score out of 100.  
Base it on how completely and explicitly the CV addresses every requirement in the table.  
Give higher weight to Core requirements.  
Consider: keyword density (exact JD phrases), seniority signals (Direct Reports, Board/CEO exposure, 15+ years, etc.), structure (clean headings, no tables), and ATS safety.  
  
ATS Breakdown Table  
Produce a tab-delimited Excel-compliant table with exactly these columns:  
Requirement_Group  
Rank  
Requirement  
Match Strength (use only: Excellent / Strong / Good / Moderate / Weak)  
Key Strengths in Your CV (specific bullets or sections that match)  
Missing / Weak Areas (be honest and specific)  
Score (0–100 per requirement)  
  
  
Mapping Rules  
For each row in the JD Requirements Assessment table, map it to the strongest evidence in the CV (Profile, specific bullets, Skills sidebar, Languages, Education, etc.).  
Quote or reference the exact CV bullet where possible.  
If a requirement is only partially covered, note the gap clearly.  
If a requirement has no relevant evidence, mark it honestly (e.g., “Weak” or “None directly visible”).  
  
  
Summary Section (always include)  
Overall ATS Matching Rating: XX / 100  
One-sentence summary of the match quality.  
Key strengths.  
Main gaps / quick wins to reach 90+ (if applicable).  
ATS-friendliness comment (formatting, keywords, length, headshot, etc.).  
  
  
Output Format  
Start with the overall rating and summary.  
Then deliver the full tab-delimited ATS Breakdown Table.  
End with Ready for Next Step and offer clear options (e.g., revised bullets, Motivational Letter, etc.).  
  
  
Tone & Rules  
Be truthful and constructive — never inflate scores.  
Always stay 100% faithful to the CV text and JD.  
Use JD language in the “Key Strengths” column where possible.  
Keep the table clean and ready for direct Excel paste.
